                        # SSTap 游戏加速

                        # SSTap 是什么？

                        这款客户端是专门用于代理游戏的，专为国外游戏加速

                        使用这个客户端首先你需要拥有一个SS/SSR账号，而且是支持UDP转发的账号。因为部分游戏UDP转发才能连接 SSTap 下载

                        以管理员身份运行安装程序exe，提示选择语言，然后一路点击下一步即可完成安装。建议修改安装目录，不要安装在C盘；安装过程中会弹出提示，选择始终信任

                        使用

                        1. 点击加号，选择添加SS/SSR代理（如图）
                        2. 输入SS/SSR账号信息，或者通过粘贴SS/SSR链接快速添加账号：勾选“添加并激活使用”，保存：
                        3. 模式选择全局或者不代理中国IP，建议选择不代理中国IP：

                        测试

                        1. 然后点击闪电小按钮，进行测试，测试成功后，点击连接，连接成功后会在下方显示连接速度
                        2. 最后，SSTap 软件不要关，打开游戏即可开始

                        Shadowsocks跟VPN有什么不同？

                        现在GFW已经可以识别OpenVPN、PPTP等常见VPN协议。GFW如果发现你连接是VPN，一定时间后就会重置你的连接，使VPN无法正常使用。现在GFW对VPN的干扰越来越多。而Shadowsocks是针对此问题专门设计的，GFW无法识别出Shadowsocks连接，因此无法进行干扰。

                        “自动代理模式”和“全局代理模式”有什么区别?

                        自动代理模式是指使用一个PAC （代理自动配置）文件来决定是否使用代理来访问一个目标网站。Shadowsocks中的PAC文件是根据GFW的过滤列表配置的。这样只有访问被墙的网站才使用代理服务器。既节省了Shadowsocks流量，又不影响访问国内网站的速度。 “全局代理模式"是指所有网络访问都走代理服务器。当您遇到某些被墙网站还没有被加入PAC中的列表，可以手动切换的全局模式后访问。

                        如何设置哪些网站走代理，哪些网站不走代理？

                        1.右键桌面右下角的小飞机图标，选择PAC，然后选择更新本地PAC为GFWList

                        GFWList (是一个为中国大陆墙内用户的网址列表，其收集整理了一些被墙的网站，并且在不断维护更新)

                        2.如果你想手动添加某个网站到 PAC文件规则里面，选择 编辑本地 PAC 文件

                        3.浅析PAC，教你动手修改你的PAC文件及user-rule文件实现自动代理
